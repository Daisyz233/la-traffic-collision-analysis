# Notebooks

Place cleaned Jupyter notebooks in this folder.

Recommended naming structure:

```text
01_data_cleaning.ipynb
02_exploratory_data_analysis.ipynb
03_spatial_analysis.ipynb
04_random_forest_modeling.ipynb
05_young_female_subgroup_analysis.ipynb
```

Before uploading notebooks to GitHub:

- Restart the kernel and run all cells.
- Remove unnecessary testing cells.
- Add markdown explanations before important code sections.
- Keep outputs that support the portfolio story, especially final charts and model results.
- Avoid uploading private file paths or local machine information.
